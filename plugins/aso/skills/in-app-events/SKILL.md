## Methodology

**In-App Events** — Self-serve App Store featuring with automatic re-engagement reach.

**Key Advantage:** Lapsed users who haven't opened the app in a while receive automatic push notifications about your event — **no push permission required**. This makes events one of the most powerful re-engagement tools available without a marketing budget.

### Where Events Appear

- Today tab (high visibility, requires 7+ day events with strong images)
- Search results (for relevant keyword searches)
- App product page (dedicated Events section)
- Personalized recommendations (Apple algorithm sends to likely-interested users)
- Automatic notification to lapsed users (the unique advantage)

### Event Types

| Type | Use Case | Example |
|------|---------|---------|
| Challenge | User participation goal | "30-Day Meditation Challenge" |
| Competition | Ranked or scored competition | "Weekly Leaderboard Sprint" |
| Live Event | Real-time event users attend | "Live Coaching Session" |
| Major Update | New feature announcement | "All-New Workout Builder" |
| Premiere | First availability of content | "Season 2 Now Available" |
| Special Event | Any other time-limited moment | "Holiday Sale — 40% Off" |

### Event Card Specs

| Field | Limit | Tips |
|-------|-------|------|
| Event name | 30 chars | Lead with user benefit, not app name |
| Short description | 50 chars | Appears in search results — include keywords |
| Long description | 120 chars | More detail, visible on event card |
| Event image | 2160×1080px (2:1) | High contrast, bold at thumbnail size |
| Duration | Up to 31 days | Minimum 7 days for Today tab consideration |
| Max simultaneous | 10 events | — |

### Planning Workflow

**Step 1: Event Idea Selection**

Match event type to app category:
- Fitness/Health → Challenges and Competitions (highest engagement)
- Education → Premiere (new course/content)
- Games → Competition (leaderboard) or Live Event
- Productivity → Major Update (feature launches)
- Any app → Special Event (seasonal promotions)

Identify primary goal:
- Re-engagement (lapsed users) → any type, focus on Apple's automatic notification
- Acquisition (new users) → search visibility is key, pick keyword-rich name
- Feature launch → Major Update type

**Step 2: Write Event Copy**

Name formula: `[User Benefit or Outcome] - [Timeframe]`
- ✅ "Hit Your Sleep Goal — 7 Days"
- ❌ "AppName Sleep Challenge"

Short description (50 chars — appears in search):
- Include 1–2 target keywords naturally
- Communicate the benefit or offer

Long description (120 chars):
- Expand on what users will get or experience

**Step 3: Event Card Image**

Design principles:
- High contrast — must work as a small thumbnail
- Bold, single visual communicating the outcome/reward
- Show results, not app UI (happy user, achievement badge, etc.)
- Text overlay with event name (large, legible)
- No small text — it won't be readable

**Step 4: Submit in App Store Connect**

Timeline:
- Submit 3–5 days before desired start date
- Apple review typically 1–3 business days
- Set end date accurately — events should end when the event ends

### Keyword Visibility

Event names and short descriptions are indexed by App Store search.
- Include 1–2 relevant keywords naturally in the event name or short description
- Don't keyword-stuff — Apple editors review these and poorly written events get rejected
- Seasonal events (e.g., "New Year Challenge") can temporarily boost keyword rankings

### Optimization Cadence

Run at least **one event per month** to:
- Maintain algorithmic placement in recommendations
- Ensure consistent lapsed-user re-engagement pushes
- Signal to Apple that the app is actively maintained

### Common Mistakes

| Mistake | Correct Approach |
|---------|-----------------|
| App name in event name | Lead with user benefit |
| Event shorter than 7 days | Minimum 7 days for Today tab consideration |
| Generic image (app screenshots) | Bold, outcome-focused custom image |
| No keywords in short description | Include 1–2 natural keywords |
| One event per quarter | At least one per month for sustained placement |
